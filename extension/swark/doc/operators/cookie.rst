cookie
------

Summary
~~~~~~~
Returns the value of the HTTP cookie identified by the parameter.

Usage
~~~~~
::

    cookie( name )

Parameters
~~~~~~~~~~
    =========== =============================================================== ======== =======
    Name        Description                                                     Required Default
    =========== =============================================================== ======== =======
    name        The name of the cookie                                          Yes
    =========== =============================================================== ======== =======

Examples
~~~~~~~~
::

    {cookie( 'login' )}

Returns the value of the cookie named login.
